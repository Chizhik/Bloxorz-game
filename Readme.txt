bloxorz game written on Scala.
cs109ui.jar (author: Otfried Cheong) file should be moved to lib folder (C:\Program Files\scala-2.11.7\lib)
Then bloxorz.scala file should be compiled (ex: fsc bloxorz.scala) and run (ex: scala Bloxorz).
This project is based on practice problem http://otfried.org/courses/cs109/project-bloxorz.html
